# Tingxie App 用户使用手册

欢迎使用 **Tingxie (听写)** 练习系统！本手册将帮助您快速了解并掌握本软件的使用。

## 1. 软件简介
Tingxie 是一款帮助学生高效练习中英文听写的 Web 应用。它不仅能播报单词，还能根据遗忘曲线提醒你进行复习，并支持手写练习。

## 2. 快速上手
### 2.1 登录与首页
- 访问应用地址 (例如：`https://tingxie.tianlu.space`)。
- 首页会显示**当前的听写任务**以及**今日需要复习的单词数量**。

### 2.2 开始听写练习
1. 在首页选择一个“周次”（如：Week 4）。
2. 点击“开始听写”。
3. 听到单词播报后，在中央书写区写下对应的答案。
4. 点击“检查”或“下一个”。系统会自动判定对错并记录进度。

### 2.3 每日复习 (今日必做)
- 每天登录后，优先点击“**今日复习**”。
- 这里出现的单词是你之前写错的，或者根据科学计算认为你快要忘记的词。
- 坚持完成每日复习是掌握词汇的关键。

## 3. 核心功能说明
### 3.1 手写识别
- 练习界面提供了一个画布区域，支持使用手指、Apple Pencil 或鼠标进行书写。
- 写完后，系统会尝试识别您的字迹并与标准答案对比。

### 3.2 错题本
- 您可以随时在菜单中进入“错题记录”，查看历史练习中出错的词汇。
- 您可以针对这些错题进行专项强化练习。

### 3.3 语言切换
- 系统支持中文和英文两种模式，不仅可以练习汉字听写，也可以练习英文单词拼写。

## 4. 常见问题
- **听不到声音？** 请检查设备是否开启静音模式，并确保浏览器允许自动播放音频。
- **字体显示不正常？** 建议使用最新版本的 Safari 或 Chrome 浏览器。
- **进度没有保存？** 系统会自动实时保存进度，请确保网络连接正常。
